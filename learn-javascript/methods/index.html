Array Methods

Push

Push adds an element to the end of an array

Pop

Pop returns and removes the last element of an array. I will store it wherever we tell it to. If we just call arr.pop without assigning it a variable the 'popped' element gets tossed out.

Pop removes from the end of an array.

link to good example

Shift

Shift is like pop, but rather than removing the last element from an array, shift removes the first.

Un-Shift

Adds to the front of an array (basically opposite of push)

Index Of

Index of tells us where the element is located in an array

Good to know. Quirk of index. -1 means that the element does not exist

Splice

When splicing we are taking out an element at the splice location. Commonly used to remove an element in the middle of an array. (and/or replace it with something else) The delete count can be 0. After the start # and deleteCount # (if you continue making a comma everything after the first two it's optional. these extra mean that we will add in this element)

Slice

arr.slice creates a new array. it is non-destructive because it just copies the elements in an array into a new one. Good way to make a copy of the array.

Sort

Will sort your array elements into numerical and alphabetical order.

Reverse

Will sort your array into the numerical and alphabetical order reversed. However, sometimes reversed will apply to the array even if you don't tell it too? IDK that seems weird but more later apparently.

Join

takes an array and returns a string

Split

breaks a string up into an array
